What is grid based design?

Designing pages by splitting the page into columns and using these to decide on the placement of elements.

Whats all the hype about responsive webpages?

With all the different smartphones and tablets on the market it's really important for webpages to work on these different devices. A responsive webpage makes the content fit right on all devices by hiding certain elements or resizing others and so on to make sure it's a good experience for the user.
